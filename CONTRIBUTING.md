# Contributing

欢迎贡献！

## 本地开发（仅 Windows）

```bash
npm install
npm start
```

## 提交 PR

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/foo`)
3. 提交改动 (`git commit -am 'Add foo'`)
4. 推送 (`git push origin feature/foo`)
5. 创建 Pull Request

## 代码规范

- 保持简洁
- 不要添加不必要的注释
- 遵循现有代码风格

## 平台说明

- 本项目仅支持 Windows
- 所有代码应保持 Windows 兼容性
- 构建命令：`npm run build`（生成 NSIS 安装包和便携版）
