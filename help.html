MinBrowser 帮助

版本 1.1.1 | 基于 Electron + Chromium 内核

基本操作

  • 在地址栏输入网址，按 Enter 或点击「前往」按钮打开网页
  • 点击 ◀ ▶ 按钮前进或后退浏览历史
  • 点击 ↻ 按钮刷新当前页面
  • 点击 ? 按钮打开本帮助页面

默认首页

默认打开必应中国搜索（cn.bing.com），启动即搜。

键盘快捷键

  • F5 或 Ctrl+R 刷新页面
  • Alt+← 后退
  • Alt+→ 前进
  • Ctrl+L 或 F6 聚焦地址栏
  • F11 全屏模式

关于 MinBrowser

MinBrowser 是一个极简开源的 Windows 浏览器，基于 Electron 构建，使用 Chromium 内核，轻量、快速、无广告。

MinBrowser Help

Version 1.1.1 | Built on Electron + Chromium

Basic Usage

  • Type a URL in the address bar, press Enter or click Go to navigate
  • Click ◀ ▶ to go back or forward in history
  • Click ↻ to reload the current page
  • Click ? to open this help page

Default Homepage

Opens Bing China (cn.bing.com) by default — search right away on startup.

Keyboard Shortcuts

  • F5 or Ctrl+R Reload page
  • Alt+← Go back
  • Alt+→ Go forward
  • Ctrl+L or F6 Focus address bar
  • F11 Full screen

About MinBrowser

MinBrowser is a minimal open-source Windows browser built on Electron with the Chromium engine. Lightweight, fast, and ad-free.

  • GitHub：https://github.com/YANBKmonster/min-browser
  • License：MIT License
